## 🎯 About

CyberLearn is a **full-stack gamified cybersecurity learning platform** deployed live at [cyberlearn-beta.vercel.app](https://cyberlearn-beta.vercel.app/). It provides an interactive web interface for learners to study cybersecurity topics, upload and access study materials, and progress through structured content — all backed by a dedicated REST API.

## 🛠️ Tech Stack

| Layer | Technology |

| **Frontend** | JavaScript (React), CSS, HTML |
| **Backend** | Node.js REST API |

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/ananyaprasa/Cyberlearn.git
cd Cyberlearn
```

**2. Set up the backend**

```bash
cd cyberlearn-backend
npm install
npm start
```

**3. Set up the frontend**

```bash
cd Frontend
npm install
npm start
```

The app will be running at **http://localhost:3000**
